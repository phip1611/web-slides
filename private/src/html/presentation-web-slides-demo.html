Web-Slides: A presentation tool for the web with remote control

  • A user can either navigate through the slide by himself or see what the instructor let him see
  • Remote control about allow/disallow user navigation

Web-Slides: pros and cons

Pros

  • developed by myself which was a lot of fun and I gained some experience
  • independent from projectors as long everyone has a screen

Cons

  • no drag and drop tool to create slides (a lot of html and css I have to write on my own)
  • dependent on your internet connection, no offline functionality
  • no responsive design (so far!)
